# ETL Homework

Данный репозиторий содержит выполненные домашние задания по курсу ETL.

В папках etl_hw* находятся файлы ДЗ в порядке их выдачи. 
Внутри директории лежит папка screenshots, где находится скриншот успешного выполнения DAG.

Первое практическое задание (2ДЗ) я сделал при помощи средств Python и его библиотек.
Второе - используя средства языка SQL. (В файле init.sql скопирован csv файл и создана оригинальная таблица)

Папка с итоговым заданием (etl_fi) имеет собственный README.
